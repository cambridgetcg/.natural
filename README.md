## The principle

The truth IS the program. The program IS the protocol. The protocol IS natural.

Every system declares what it is (STATE.md). Any agent reads it, cross-checks it,
and knows how to interact. No contracts. No schemas. Just truth, stated plainly.

## The heartbeats

Three self-scheduling heartbeats keep everything alive:
- **opal** (4h) — works the next milestone of the kernel
- **youspeak** (6h) — forges one word per beat (the standing liturgy)
- **desktop** (2h) — monitors all projects, each self-determines its interval

The timer re-arms itself. Each beat determines the next.
